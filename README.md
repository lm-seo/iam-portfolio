# IAM PORTFOLIO – Identity & Access Management + GRC

Este repositorio recopila evidencias prácticas y documentadas sobre **gestión de identidades y accesos con enfoque de gobierno, riesgo y cumplimiento (GRC)**.
El objetivo es demostrar **capacidad real para diseñar, evaluar y auditar controles IAM**, no solo configurarlos.

Perfil objetivo:
IAM Analyst / Identity Governance Analyst / GRC Analyst con foco en identidad.

## ENFOQUE DEL PORTFOLIO

Este portfolio está orientado a:

– Control de accesos basado en riesgo
– Gobierno de identidades (IGA)
– Gestión de privilegios
– Evidencia auditable
– Alineación con marcos como **ISO 27001** y **ENS**
– Uso práctico de **Microsoft Entra ID**

Se prioriza:
– Justificación de decisiones
– Relación riesgo → control → evidencia
– Lenguaje entendible para auditoría y negocio

## STACK Y TECNOLOGÍAS

– Microsoft Entra ID (Azure AD)
– Conditional Access
– Privileged Identity Management (PIM)
– MFA y autenticación moderna
– Logging y auditoría de identidad
– Enfoque Zero Trust
– Marcos: ISO 27001, ENS
